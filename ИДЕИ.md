# Идеи для новых скриптов

## Изображения
- **Генератор иконок** — из одного PNG сразу набор: 16, 32, 64, 128, 256, 512px + .ico
- **Собрать коллаж** — несколько картинок в одно изображение: горизонтально, вертикально или сеткой N×M
- **Оптимизировать для веба** — сжать PNG/JPG без видимой потери качества
- **Извлечь цветовую палитру** — показать 5–10 доминирующих цветов с HEX-кодами

## PDF
- **Объединить PDF** — склеить несколько файлов в один (ghostscript)
- **PDF → изображения** — каждую страницу в PNG/JPG нужного разрешения
- **Изображения → PDF** — собрать несколько картинок в один PDF

## Видео
- **Вырезать фрагмент** — указать начало и конец, вырезать без перекодирования
- **Сделать GIF** — с настройками fps, размера и количества цветов
