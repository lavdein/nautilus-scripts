# gh0st.tools

Скрипты для контекстного меню GNOME Nautilus. Устанавливаются в `~/.local/share/nautilus/scripts/`.

## Видео

| Скрипт | Что делает | Зависимости |
|--------|-----------|-------------|
| Извлечь звук | Вытаскивает аудиодорожку в mp3 / aac / flac / ogg / wav | `ffmpeg` |
| Информация о видео | Показывает кодеки, разрешение, битрейт, длительность | `ffmpeg` |
| Сжать видео | Перекодирует в H.264 с выбором пресета качества | `ffmpeg` |
| Сменить контейнер (без потерь) | Перепаковывает в mp4 / mkv / mov / avi без перекодирования | `ffmpeg` |

## Изображения

| Скрипт | Что делает | Зависимости |
|--------|-----------|-------------|
| Изменить размер | Масштабирует по %, ширине, высоте или пресету | `imagemagick` |
| Конвертировать в формат | Конвертирует в png / jpg / webp / avif / gif | `imagemagick` |

## Утилиты

| Скрипт | Что делает | Зависимости |
|--------|-----------|-------------|
| Открыть терминал здесь | Открывает Ptyxis в текущей папке | `ptyxis` |
| Транскрибировать (Whisper) | Транскрибирует аудио/видео в txt и/или srt через faster-whisper | Python venv (см. ниже) |

## Файлы

| Скрипт | Что делает | Зависимости |
|--------|-----------|-------------|
| Копировать путь | Копирует полный путь файла(ов) в буфер обмена | `wl-clipboard` или `xclip` |
| Создать архив | Упаковывает выбранные файлы в zip / tar.gz / tar.xz | `zip`, `tar` |

## Установка Whisper

После клонирования репо или переустановки системы:

```bash
bash ~/.local/share/nautilus/scripts/.meta/install.sh
```

Создаёт venv с `faster-whisper` (CUDA). Модели скачиваются автоматически при первом запуске.
